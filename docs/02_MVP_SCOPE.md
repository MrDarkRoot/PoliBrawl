# MVP Scope

This document freezes the MVP scope. Anything outside this list is deferred unless explicitly re-approved.

## Build In MVP

- Landing page
- Platform list
- Platform survival page
- Admin platform registry
- Admin source registry
- Fetch URL / paste text
- Keyword scanner
- Candidate review queue
- Red flag editor
- Evidence builder
- Survival note editor
- Backup option editor
- Checklist editor
- Publisher
- Request review form
- Watch platform email capture
- Submit experience form
- Suggest survival tip form
- Correction form
- Most requested leaderboard

## Do Not Build In MVP

- forum
- public comments
- DM
- user profile feed
- full crawler
- AI legal scanner
- PDF generator
- subscription
- browser extension
- API
- enterprise dashboard
- complex document versioning

## MVP Interpretation Rules

- Human review is required before public publication.
- Sources are editorial inputs, not the end-user product.
- Community input is structured intake, not a public discussion system.
- Public pages should optimize for clarity, search intent, and trust.
- If a feature adds complexity without helping the founder publish survival pages faster, cut it from MVP.
