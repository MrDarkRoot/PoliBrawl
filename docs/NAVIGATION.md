# Navigation

This document defines admin navigation for Epic B and future public navigation for Epic C.

## Admin Navigation

Epic B admin nav:
- Dashboard
- Platforms
- Sources
- Candidates
- Red Flags
- Evidence
- Survival Notes
- Backup Options
- Checklists
- Community
- Publisher
- Settings later

## Admin Dashboard Widgets

- platforms needing review
- pending candidates
- red flags missing evidence
- pending community submissions
- stale platform pages
- most requested reviews

## Future Public Navigation

Epic C public nav later:
- Home
- Platforms
- Leaderboards
- Methodology
- Submit Experience
- Request Review

## Rules

- legacy clause, rule, and signal navigation should be hidden or separated as Legacy
- do not keep legacy navigation dominant
- admin preview URLs must not be public or indexed
- public navigation should prioritize platform pages and trust-building pages over social surfaces
